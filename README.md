# eBook-mongo-node-react-multer
# EBook

## technology
- Back :
    - Mongo (data base), Mongoose
    - Express (server)
    - Node JS
    - multer (to upload file)
- Front :
    - React
    - Recoil (state)
    - Boostrap


## install
- run mongo

- add file .env
    `PORT=5050
     MONGO_URL=mongodb://localhost:27017/biblio`

- data base : Mongo
    - name : "biblio"

- server : (with nodemon)
    - npm install
    - npm start

- client :
    - yarn
    - yarn start
