## Getting Started

### Prerequisites

* node
* yarn

### Installation

1. Clone the repo
```sh
git clone https://github.com/decuyperanthony/Ebook-app-MERN-uploadFile
```

2. Database
```sh
database : Mongo
name : "biblio"
```
2. To init server
```sh
cd server
cp .env.example .env
npm install
npm start
```
3. To init client
```sh
cd client
yarn
yarn start
```
